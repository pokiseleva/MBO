knapsack.py -- генерация случайных задач о рюкзаке размерности от 5 до 20, с весами и ценностями из промежутка [0,1]. 
Решение задачи с помощью динамического программирования.

knapsack_problems.txt -- случайно сгенерированная база данных задач размерности от 5 до 20.

solution_knapsack_problems.txt -- результат работы программы knapsack.py решения случайно сгенерированных задач knapsack_problems.txt.

knapsack_data.txt -- база данных задач о рюкзаке с сайта Kaggle.

data_conversation.py  -- преобразование данных с сайта Kaggle в данные для программы knapsack_problem.py. 

data_result.py -- преобразование данных с сайта Kaggle в данных с решениями задач о рюкзаке.

knapsack_data_set.txt -- преобразованные данные с сайта Kaggle с решениями. 

knapsack_data_result.txt - данные с сайта Kaggle преобразованные в данные для программы knapsack_problem.py; 

knapsack_data.py - решение задачи о рюкзаке полным перебором;

UI.py -- генерация базы данных некоррелированных задач;

UI.txt -- база данных некоррелированных задач о рюкзаке с сайта Kaggle. 

UI_output.txt-- некоррелированные данные с сайта Kaggle преобразованные в данные для программы knapsack_data.py; 

UI_data_result.txt -- результат работы программы knapsack_data.py решения базы некоррелированных задач с сайта Kaggle;

solution_MBO_data_UI.txt -- результат работы программы MBO_knapsack.py решения базы некоррелированных задач с сайта Kaggle;

deviation_UI_data.txt -- результат работы программы deviation.py, которая считает процент отклонения точного решения базы некоррелированных задач с сайта Kaggle от приближенного решения;

UI_problems.txt -- база данных некоррелированных задач.

UI_result.txt -- результат программы UI.py решения базы данных некоррелированных задач UI_problems.txt.

WCI.py -- генерация базы данных слабо коррелированных задач.

WCI_problems.txt -- база данных слабо коррелированных задач.

WCI_result.txt -- результат программы WCI.py решения базы данных некоррелированных задач WCI_problems.txt.

SCI.py -- генерация базы данных сильно коррелированных задач.

SCI_problems.txt -- база данных сильно коррелированных задач.

SCI_result.txt -- результат программы SCI.py решения базы данных некоррелированных задач SCI_problems.txt.

ISCI.py -- генерация базы данных обратно сильно коррелированных задач.

ISCI_problems.txt -- база данных обратно сильно коррелированных задач.

ISCI_result.txt -- результат программы IWSK.py решения базы данных некоррелированных задач ISCI_problems.txt.

MBO_knapsack.py -- алгоритм оптимизации Monarch butterfly optimization.

solution_MBO_knapsack_problems.txt -- результат работы алгоритма MBO решения задачи о рюкзаке из базы (размерность от 5 до 20) knapsack_problems.txt.

solution_MBO_UI.py -- решение задачи о рюкзаке с помощью алгоритма MBO базы данных некоррелированных задач UI_problems.txt.

solution_MBO_WCI.py -- решение задачи о рюкзаке с помощью алгоритма MBO базы данных слабо коррелированных задач WCI_problems.txt.

solution_MBO_SCI.py -- решение задачи о рюкзаке с помощью алгоритма MBO базы данных сильно коррелированных задач SCI_problems.txt.

solution_MBO_ISCI.py -- решение задачи о рюкзаке с помощью алгоритма MBO базы данных обратно сильно коррелированных задач ISCI_problems.txt.

deviation.py -- подсчет процента отклонения оптимального решения задачи о рюкзаке от решения, полученного с помощью алгоритма MBO.

deviation_UI.txt -- результат работы программы deviation.py, которая считает процент отклонения точного решения базы некоррелированных задач от приближенного решения.

deviation_WCI.txt -- результат работы программы deviation.py, которая считает процент отклонения точного решения базы слабо коррелированных задач от приближенного решения.

deviation_SCI.txt -- результат работы программы deviation.py, которая считает процент отклонения точного решения базы сильно коррелированных задач от приближенного решения.

deviation_ISCI.txt -- результат работы программы deviation.py, которая считает процент отклонения точного решения базы обратно сильно коррелированных задач от приближенного решения.

WCI_graph.py -- построение графика зависимости размерности от процента отклонения для слабо коррелированных задач.

ISCI_graph.py -- построение графика зависимости размерности от процента отклонения для обратно сильно коррелированных задач.

SCI_graph.py -- построение графика зависимости размерности от процента отклонения для сильно коррелированных задач.

UI_graph.py -- построение графика зависимости размерности от процента отклонения для некоррелированных задач.

table.py -- построение таблицы зависимости размерности от процента отклонения для каждого типа задач.
